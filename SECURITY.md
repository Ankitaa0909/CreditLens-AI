# Security Policy

## Supported Versions

This project is currently developed for educational and hackathon purposes.

## Reporting Security Issues

If you discover a security vulnerability, please create a GitHub Issue or contact the project maintainer.

## Data Privacy

- No real customer banking information is stored.
- All datasets are simulated.
- No production banking APIs are used.

Future releases will include:

- Encryption
- Role Based Access Control
- Audit Logging
- Secure API Authentication
